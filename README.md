# Easy MC Server



Easy MC Server es una aplicación de escritorio para gestionar tus servidores de Minecraft desde una interfaz gráfica sencilla e intuitiva.

![screenshot.png](screenshot.png)

## Como se usa

1. Abre la aplicacion.
2. Crea un servidor nuevo o importa uno existente.
3. Selecciona el servidor en la lista de la izquierda.
4. Inicia y detiene el servidor desde el panel de control.
5. Usa la consola para ver logs y enviar comandos.
6. Gestiona usuarios seleccionándolos y pulsando click derecho.
7. Ajusta configuraciones del servidor desde los paneles de la izquierda.

## Funciones principales

- Crear e importar servidores.
- Gestionar varios servidores desde una sola ventana.
- Ver consola en tiempo real.
- Gestión de usuarios activos.
- Abrir carpeta del servidor rapidamente.
- Editar opciones basicas del servidor.
